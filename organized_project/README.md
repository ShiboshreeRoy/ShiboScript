# Organized ShiboScript Project Files

This directory contains all the files that were moved from the root directory during project organization.

## Directory Structure

```
organized_project/
├── core/                 # Core Python source files
├── examples/             # All example scripts (.shibo files)
│   ├── examples_original/ # Original examples directory
│   └── all_shibo_files/  # All .shibo files
├── docs/                 # Documentation files (.md files)
├── tests/                # Test files
├── assets/               # Asset files (icons, images, html files, etc.)
├── legacy/               # Legacy files, backups, and unused files
└── README.md            # This file
```

## Contents Summary

### core/
Contains the main Python source files of the original project structure.

### examples/
Contains all the .shibo example files demonstrating various ShiboScript features and capabilities.

### docs/
Contains all the Markdown documentation files from the original project.

### assets/
Contains various asset files like icons, images, and HTML files.

### legacy/
Contains backup files, distribution artifacts, and other legacy files that are no longer actively used.

## Purpose

This organization helps maintain a clean root directory while preserving all original files for reference or future use.
